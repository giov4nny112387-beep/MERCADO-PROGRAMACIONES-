☁️ Google Sheets — Configuración de Sincronización

Pega aquí la URL de tu Google Apps Script Web App. La programación se guardará automáticamente en la pestaña PROG_TIENDA de tu Google Sheets cada vez que generes o modifiques el horario.

¿Cómo obtener la URL?
1. Abre tu Google Sheets → Extensiones → Apps Script
2. Borra todo y pega el código del archivo google-apps-script.js
3. Clic en Implementar → Nueva implementación
4. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario
5. Copia la URL generada y pégala aquí
⚠️

¿Desea cargar nuevos datos?

Esta acción descargará la programación desde Google Sheets.

⚠️ ADVERTENCIA: Si ya tiene una programación activa, todos los datos actuales serán reemplazados por los datos descargados desde Google Sheets.

Las rotaciones pendientes, cambios sin guardar y ajustes manuales se perderán de forma permanente.
